# 테이블오더 서비스 - 페르소나

## Persona 1: 고객 (Customer)

- **이름**: 고객 (Customer)
- **역할**: 매장에서 식사하는 고객. 테이블에 비치된 태블릿을 통해 메뉴를 확인하고 주문함.
- **주요 목표**:
  - 직원 호출 없이 빠르고 편리하게 주문하기
  - 메뉴 정보(이름, 가격, 설명, 이미지)를 충분히 확인한 후 주문하기
  - 현재 주문 내역과 상태를 실시간으로 확인하기

## Persona 2: 관리자 (Admin)

- **이름**: 관리자 (Admin)
- **역할**: 매장 운영자. 관리 화면을 통해 주문을 모니터링하고 매장을 관리함.
- **주요 목표**:
  - 실시간으로 들어오는 주문을 빠르게 확인하고 처리하기
  - 테이블별 주문 현황을 한눈에 파악하기
  - 메뉴 정보를 손쉽게 등록/수정/삭제하기
  - 테이블 세션을 관리하여 고객 회전율을 효율적으로 운영하기
